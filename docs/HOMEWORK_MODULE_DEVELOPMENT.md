### 📋 使用说明

#### 1. 安装依赖（解决网络问题后）
```bash
# 使用国内镜像
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple django djangorestframework

# 或使用离线包
pip download django djangorestframework django-filter drf-yasg pillow
pip install *.whl
```

#### 2. 数据库迁移
```bash
python manage.py makemigrations homework
python manage.py migrate
```

#### 3. 运行测试
```bash
python manage.py test apps.homework
```

#### 4. 启动开发服务器
```bash
python manage.py runserver
```
