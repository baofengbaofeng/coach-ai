## 🎯 项目概述

### 项目目标
基于DDD+EDA架构，开发面向中国家庭的智能教育管理SaaS平台，实现作业批改和运动计数两大核心功能，完成MVP版本开发。

### 技术栈
- **后端**：Python 3.12 + Tornado + MySQL 5.8
- **前端**：Vue 3纯脚本（无需Node.js编译）
- **架构**：DDD（领域驱动设计）+ EDA（事件驱动架构）
- **部署**：前后端统一部署，Docker容器化

### 开发原则
1. **模块化开发**：按DDD领域划分模块，独立开发测试
2. **渐进式实现**：从核心功能开始，逐步扩展
3. **自动化测试**：每个模块必须包含单元测试和集成测试
4. **代码规范**：严格遵循`.rules/coding-style.md`规范
5. **中文注释**：所有代码必须包含中文注释
