## 🎯 产品概述

### 产品定位
CoachAI 是一个智能教育管理SaaS平台，通过硬件外设（摄像头、麦克风）和AI技术（图像识别、动作识别）帮助家长管理子女的学习和运动。系统采用多租户架构，每个家庭作为一个独立租户，支持家庭成员角色定义和管理。产品仅支持移动端H5访问，深度集成摄像头（作业图像采集、动作识别）和麦克风（语音交互），为家庭教育管理提供智能化解决方案。

### MVP核心功能范围
1. **作业批改**：支持口算题自动批改
2. **知识总结**：自动生成学习知识总结
3. **学习计划**：个性化学习计划制定
4. **任务待办**：学习任务管理和待办事项
5. **运动识别**：支持跳绳和拍球识别
6. **运动计数**：跳绳个数、拍球个数自动计数

### 关键特性
1. **SaaS多租户模式**：以家庭为单位，支持家庭成员管理
2. **移动端H5优先**：仅支持移动端H5，优化移动体验
3. **硬件深度集成**：摄像头（作业图像采集、运动识别）+麦克风（语音交互）
4. **教育场景优化**：针对家庭教育管理场景的特别功能设计

### 核心价值
1. **作业批改自动化** - 通过摄像头采集作业图像，AI自动批改和分析
2. **运动计数智能化** - 通过摄像头识别动作，自动计数和姿势指导
3. **学习管理科学化** - 系统化的学习计划、知识总结、任务管理
4. **家长监督便捷化** - 家长全面掌握子女学习和运动情况
5. **亲子互动增强** - 家长和孩子通过系统互动，改善亲子关系

### 目标用户（以家庭为单位）
#### 家庭类型
1. **核心教育家庭** - 父母+K12子女（主要目标，6-15岁）
2. **重视教育家庭** - 关注子女教育的家庭
3. **双职工家庭** - 父母工作繁忙，需要工具辅助教育管理
4. **素质教育家庭** - 重视子女全面发展的家庭

#### 家庭成员角色
1. **家庭管理员** - 家长，购买决策者，账户管理者，主要使用者
2. **学生成员** - 子女，实际使用者，作业完成和运动执行
3. **辅助成员** - 其他家庭成员，可能协助监督或查看报告

### 技术栈
- **前端**: Vue 3 + TypeScript + Vite + PWA (移动端H5优化)
- **后端**: Python + FastAPI + SQLAlchemy + 多租户支持
- **数据库**: PostgreSQL + 多租户数据隔离
- **AI服务**: 
  - OCR识别：Tesseract/PaddleOCR（口算题识别）
  - 动作识别：MediaPipe（跳绳、拍球识别）
  - 语音识别：Whisper/百度语音API
- **硬件集成**: WebRTC（音视频）、Web Audio API、设备API
- **部署**: Docker + Nginx + 多租户资源隔离
