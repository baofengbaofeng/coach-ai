## 🎯 产品概述

### 产品定位
CoachAI 是一个面向中国家庭的智能教育管理SaaS平台，通过硬件外设（摄像头、麦克风）和AI技术（图像识别、动作识别）帮助家长管理子女的学习和运动。系统采用多租户架构，每个家庭作为一个独立租户，支持家庭成员角色定义和管理。产品仅支持移动端H5访问，深度集成摄像头（作业图像采集、动作识别）和麦克风（语音交互），为家庭教育管理提供智能化解决方案。

### 关键特性
1. **SaaS多租户模式**: 家庭为单位，支持家庭成员管理
2. **移动端H5优先**: 仅支持移动端H5，优化移动体验
3. **硬件深度集成**: 摄像头（作业图像采集、运动识别）+麦克风（语音交互）
4. **教育场景优化**: 针对家庭教育管理场景的特别功能设计

### 核心价值
1. **作业批改自动化** - 通过摄像头采集作业图像，AI自动批改和分析
2. **运动计数智能化** - 通过摄像头识别动作，自动计数和姿势指导
3. **家庭教育管理** - 家长全面掌握子女学习和运动情况
4. **亲子互动增强** - 家长和孩子通过系统互动，改善亲子关系
5. **成长数据跟踪** - 记录学习和运动数据，提供成长报告

### 目标用户（以家庭为单位）
#### 家庭类型
1. **核心教育家庭** - 父母+K12子女（主要目标，6-15岁）
2. **重视教育家庭** - 关注子女教育的城市中产家庭
3. **双职工家庭** - 父母工作繁忙，需要工具辅助教育管理
4. **素质教育家庭** - 重视子女全面发展的家庭

#### 家庭成员角色
1. **家庭管理员** - 家长，购买决策者，账户管理者，主要使用者
2. **学生成员** - 子女，实际使用者，作业完成和运动执行
3. **辅助成员** - 其他家庭成员，可能协助监督或查看报告

### 技术栈
- **前端**: React 18 + TypeScript + Vite + PWA (移动端H5优化)
- **后端**: Django 5.0 + Django REST Framework + Django Tenants (多租户)
- **数据库**: PostgreSQL 15 + 多租户数据隔离
- **AI服务**: MediaPipe (动作识别), Whisper (语音识别), 自定义模型
- **硬件集成**: WebRTC (音视频), Web Audio API, 设备API
- **部署**: Docker + Kubernetes + 多租户资源隔离
